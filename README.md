# Quantitative-Portfolio-Management
1. Data-Driven Modeling: Leveraged Python and financial APIs to ingest multi-asset historical data, calculating daily returns and covariance matrices to quantify risk-reward correlations across diversified assets.
2. Simulation & Optimization: Executed Monte Carlo simulations to visualize the Efficient Frontier and utilized the SciPy optimization engine (SLSQP) to mathematically solve for the Maximum Sharpe Ratio and Minimum Volatility weight allocations.
3. Strategic Decision Support: Developed visual analytics to compare simulated portfolios against optimized theoretical limits, providing data-backed evidence for capital allocation and risk management strategies.
